## Installation

Clone this repository

```bash
git clone https://github.com/sleepy-coder-101/React-GPT-3-UI-UX
```

Change to project directory

```bash
cd React-GPT-3-UI-UX
```

Install required NPM Packages

```bash
npm install
```

Start development server

```bash
npm start
```
